NAME
	sp_error - get a string error description

SYNOPSIS
	#include <sophia.h>

	char *sp_error(void *ptr);

DESCRIPTION
	The sp_error(3) function is used to get error description in a
	string format after any error occur.

	Environment or database handle must be used.

EXAMPLE


	void *db = sp_open(env);
	if (! db) {
		printf("error: %s\n", sp_error(env));
	}


RETURN VALUE
	On success, sp_error(3) function returns string;
	If there is no error, sp_error(3) function returns NULL;

SEE ALSO
	sp_env(3), sp_open(3)
