NAME
	sp_destroy - free any handle

SYNOPSIS
	#include <sophia.h>

	int sp_destroy(void *ptr);

DESCRIPTION
	The sp_destroy(3) function is used to free memory used by
	any sophia handle, such as: environment, database or cursor.

EXAMPLE

	void *db = sp_open(env);
	/* ... */
	int rc = sp_destroy(db);
	if (rc == -1) {
		printf("error: %s\n", sp_error(env));
	}


RETURN VALUE
	On success, sp_destroy(3) functions returns 0;
	On error, it returns -1. See sp_error(3) for details.

SEE ALSO
	sp_env(3), sp_open(3), sp_cursor(3)
